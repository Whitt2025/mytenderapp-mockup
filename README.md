# MyTenderApp — Tender Profiles

A proof-of-concept demo for MyTenderApp, a three-sided marketplace connecting bartenders (Tenders), patrons, and venues in the nightlife industry.

## What This Is

This is a working demo/proof-of-concept built in HTML and hosted on GitHub Pages. It showcases the core experience of the app including:

- Tender profile creation with photo upload
- Patron profile creation
- Venue (Place) profile creation
- Tinder-style swipe deck to discover and follow Tenders and Places
- Browse directory of all Tenders, Patrons, and Places
- Tender and Place owner dashboard (PIN protected)
- Admin dashboard for partners
- Affiliate offers page
- Home dashboard with live shift status
